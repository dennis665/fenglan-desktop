# 2D 桌面魔法寵物 (Desktop Mascot) 使用說明書

歡迎召喚您的專屬 2D 桌面魔法寵物！桌面互動程式。

---

## 🌟 核心特色
1. **透明與穿透視窗**：角色完美立於桌面上，背景完全透明，且滑鼠移到空白透明處時會自動穿透（不影響您點擊桌面其他視窗）。
2. **重力下墜與掙扎**：用滑鼠左鍵可以抓起寵物隨意拖動，此時寵物會掙扎晃動；放開滑鼠後會因「重力加速度」掉落回螢幕底部。
3. **日常小任務與升級解鎖**：完成任務獲取能量，提升等級以解鎖更多角色、特殊動作和背景音樂！
4. **電腦狀態與趣味對話**：寵物會定時監控您的 CPU 與記憶體使用率，並在系統過載時彈出精準的趣味對話框。
5. **8-Bit 復古歌單**：內建 Web Audio API 電子樂播放器，無須網路即可播放 3 首不同情境的背景音樂。

---

## 🎮 操控指南

### 1. 互動操作
* **滑鼠左鍵點擊**：寵物會高興地跳起並播放特效（增加日常觸摸進度）。
* **滑鼠左鍵長按拖曳**：可以抓起寵物移動位置。
* **滑鼠右鍵點擊**：彈出客製化選單，包含「日常任務」、「收藏庫」、「影音播放」、「警告閥值設定」、「開發者模式」、「電腦資訊」、「離開寵物」。

### 2. 🛠️ 開發者模式 (Developer Mode)
* **功能**：在右鍵選單點擊「開發者模式」，即可直接進入進行管理：
  1. 可在選單選取指定「角色」與「動作」。
  2. 點擊上傳自訂的 `.webp` 動圖。系統會自動將該動圖複製至專案的 `assets/` 資料夾並自動套用。
  3. **動圖庫歷史保留**：所有曾上傳過的 WebP 檔案皆會被妥善保存在動圖庫歷史中，您可以隨時點選「啟用」切換回舊的 WebP 動圖！
  4. **原生動圖呈現**：圖片背景維持透明，且無任何程式控制的強制位移/抖動動畫，完全由您的 WebP 動圖自身決定如何舞動，流暢度最高！

### 3. 養成與解鎖要素
透過右鍵選單開啟「日常任務」與「收藏庫」：
* **角色切換頁**：
  * 🐱 **魔法小貓**：預設角色 (Lv.1)。
  * ➕ **自訂角色**：可以在開發者模式中自由新增任何自訂角色！輸入角色作品與角色名稱，即可立刻在收藏庫解鎖並啟用！
* **動作展示頁**：
  * **原地發呆** / **踏步走動**：預設動作。
  * **魔法轉圈圈**：Lv.2 自動解鎖。
  * **釋放小魔法**：Lv.4 自動解鎖。
* **影音播放器頁**：
  * ➕ **自訂影音上傳**：不限開發者模式，所有人皆可在「影音播放器」頁面點擊「上傳影音」自由匯入音訊或影片檔案（支援 `.mp3`, `.wav`, `.mp4`, `.webm`, `.ogg`）。
  * 🎬 **影片預覽螢幕**：上傳影片時會自動在播放器面板中展開顯示影片畫面；播放音訊時則會顯示帶有跳動音符動畫的音效背景。
  * 🔁 **雙重循環播放**：可隨時一鍵切換「單曲循環 🔂」與「全曲循環 🔁」模式。
  * 🗑️ **硬碟同步清理**：自訂影音可隨時點擊 `🗑️` 刪除，程式會同步從硬碟中將該 WebM/MP3/MP4 實體檔案刪除得一乾二淨！

### 3. 日常小任務 (每日重置)
* **每日簽到**：開啟任務面板點擊「簽到」即可完成（+20 能量）。
* **輕摸互動 5 次**：點擊小貓 5 次（+10 能量）。
* **桌面散步 100 步**：放著讓小貓在桌面隨機走動 100 步（+15 能量）。
* **聆聽影音 1 分鐘**：開啟影音播放器播放音樂或影片累積滿 60 秒（+20 能量）。
* *能量滿 100 * 等級數值時自動升級，並會彈出慶祝對話框！*

## 💻 系統警報與對話框
小貓會定時監控系統狀態並發送警報，您可以隨時在右鍵選單的**「警告閥值設定」**中拖動滑桿自訂觸發閾值（預設 CPU 為 80%，記憶體為 85%）：
* **CPU 使用率過高**：超過您設定的閾值時，例如：「*💦 主人！電腦好燙喔！目前 CPU 使用率高達 85%！小貓快熱昏了喵！*」
* **記憶體不足**：超過您設定的閾值時，例如：「*💾 記憶體快被吃光光了！目前使用率 88% (剩餘 2048MB)！快關掉一些沒用的分頁吧！*」
* **深夜提醒 (23:00 ~ 05:00)**：彈出溫馨小叮嚀，例如：「*💤 已經是深夜了呢，主人該睡覺了，熬夜傷肝喔！*」

## 🚀 下載後的安裝、測試與打包指南

當您（或其他人）從 GitHub 將此專案（推薦命名為 **`fenglan-desktop`**）下載/複製下來後，請遵循以下步驟進行安裝、測試與打包：

### 🛠️ 1. 初始化與環境變數設定
本專案支援使用 `.env` 設定檔來控制本地參數（例如：開發者模式開關）。
1. **複製範本檔案**：
   在專案根目錄下，將 `.env.example` 複製一份並更名為 **`.env`**。
2. **調整參數開關**：
   打開 `.env` 檔案，您可以根據需要修改參數：
   ```env
   # 是否啟用開發者模式選單 (true = 顯示, false = 隱藏)
   # ⚠️ 發布至 GitHub 前請維持設定為 false 以確保一般使用者無法開啟後台。
   ENABLE_DEVELOPER_MODE=true
   ```

### 📥 2. 安裝 Node.js 依賴套件
在專案根目錄下打開終端機（如 Git Bash, PowerShell, 或 CMD），執行以下指令來安裝所有必要的 Electron 開發套件：
```bash
npm install
```

### 🏃 3. 本地執行與測試 (一般啟動)
安裝完成後，您可以使用以下指令啟動程式進行測試與偵錯：
```bash
npm start
```
這會在前台打開桌寵，您可以在此時打開開發者工具進行調試。

> [!TIP]
> **Windows 背景靜默啟動**：
> 測試完畢後，若平時希望桌寵完全在背景執行，不彈出 CMD 視窗打擾您工作，可直接雙擊根目錄下的 **`啟動寵物.vbs`** 來靜默開啟程式。

### 📦 4. 打包成獨立免安裝的 EXE 檔案 (Windows 用)
如果您希望將專案打包成一個不需要安裝 Node.js 環境、雙擊即可直接執行的免安裝綠色版 EXE 檔案（方便分享給朋友或自己收藏），請執行：
```bash
npm run build
```
* **打包產出**：打包完成後，根目錄下會自動生成一個 **`dist/`** 資料夾。
* **執行檔案**：進入 `dist/Mascot-win32-x64/`，您會看到一個 **`Mascot.exe`**。直接雙擊它即可完美獨立執行您的 2D 桌面魔法寵物與影音播放器！
* **分發說明**：若要分享給他人，只需將整個 `dist/Mascot-win32-x64/` 資料夾壓縮成 `.zip` 傳給對方，解壓縮後雙擊 `Mascot.exe` 即可使用！
